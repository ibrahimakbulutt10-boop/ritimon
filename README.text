🎧 RitimON FM - Gelişmiş Radyo ve Chat Platformu

## 🚀 Özellikler

### 🎵 Radyo Özellikleri
- **Canlı Yayın**: Gerçek zamanlı ses yayını
- **DJ Paneli**: Gelişmiş DJ kontrol paneli
- **Şarkı Bilgileri**: Şu anda çalan şarkı gösterimi
- **Ses Kontrolü**: Ses seviyesi ve çalma/durdurma
- **Vinyl Animasyonu**: Gerçekçi plak animasyonu

### 💬 Chat Özellikleri
- **Gerçek Zamanlı Chat**: Socket.io ile anlık mesajlaşma
- **Kullanıcı Listesi**: Çevrimiçi kullanıcıları görme
- **Emoji Desteği**: Emoji seçici ile mesajlara emoji ekleme
- **Sistem Mesajları**: Otomatik sistem bildirimleri
- **Komutlar**: /help, /clear, /users gibi komutlar
- **Duyurular**: DJ'den özel duyurular

### 🎙️ DJ Paneli
- **Yayın Kontrolü**: Yayını başlatma/durdurma
- **Müzik Kontrolü**: Şarkı bilgilerini güncelleme
- **Duyuru Sistemi**: Dinleyicilere duyuru gönderme
- **Canlı Chat**: Dinleyici mesajlarını görme
- **İstatistikler**: Dinleyici sayısı ve yayın süresi
- **Hızlı İşlemler**: Şarkı isteği, acil durdurma

### 📱 Modern Arayüz
- **Responsive Tasarım**: Mobil ve masaüstü uyumlu
- **Glass Morphism**: Modern cam efekti tasarım
- **Gradient Renkler**: Göz alıcı renk geçişleri
- **Animasyonlar**: Yumuşak geçiş efektleri
- **Dark Theme**: Karanlık tema desteği

## 🛠️ Kurulum

### Gereksinimler
- Node.js (v14 veya üzeri)
- npm veya yarn

### Adımlar
1. **Bağımlılıkları yükleyin:**
   ```bash
   npm install
   ```

2. **Sunucuyu başlatın:**
   ```bash
   npm start
   ```

3. **Tarayıcıda açın:**
   ```
   http://localhost:3000
   ```

## 🔧 API Endpoints

- `GET /api/users` - Çevrimiçi kullanıcılar
- `GET /api/djs` - Aktif DJ'ler
- `GET /api/status` - Sunucu durumu
- `GET /health` - Sağlık kontrolü

## 🎮 Kullanım

### Dinleyici Olarak
1. Ana sayfadan "🎧 Dinleyici" butonuna tıklayın
2. Radyo otomatik olarak başlar
3. Chat'e katılmak için "💬 Sohbete Katıl" butonuna tıklayın

### DJ Olarak
1. Ana sayfadan "🎙️ DJ Paneli" butonuna tıklayın
2. DJ adınızı ve şifreyi girin (şifre: 4545)
3. Yayını başlatın ve müzik çalmaya başlayın

### Chat Kullanımı
1. "💬 Sohbete Katıl" butonuna tıklayın
2. Takma adınızı girin
3. Mesajlarınızı yazın ve emoji ekleyin

## ⌨️ Klavye Kısayolları

### Dinleyici Sayfası
- `Space` - Çal/Durdur
- `↑` - Ses seviyesini artır
- `↓` - Ses seviyesini azalt

### Chat
- `Enter` - Mesaj gönder
